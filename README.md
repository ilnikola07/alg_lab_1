Этот проект создан на основе учебной практики для решения 3-х экономических задач. Был реализован пользовательский консольный интерфейс, позволяющий выбрать 
нужную задачу для решения.
Задача 1 (базовая). Вычислить стоимость покупки со скидкой. Ввести начальную цену и процент скидки.
Задача 2 (средняя). Вычислить сумму скидки в рублях.
Задача 3 (расширенная). Определить, сколько процентов составляет итоговая цена от первоначальной. Вычислить, сколько товаров по итоговой цене можно купить на сумму первоначальной цены одного товара.
Для запуска потребуется клонировать репозиторий по https://github.com/ilnikola07/alg_lab_1.git и запустить с помощью Visual Studio 2019+, используя ЯП С# 8.0.
